# 🛍️ Product Listing App (React + TypeScript)

A modern and responsive eCommerce product listing application built using React and TypeScript. This app displays a variety of products with images, prices, and descriptions in a clean and user-friendly interface.

---

## 🚀 Features

* 🧾 Display multiple products in a responsive grid layout
* 🖼️ Product images with clean UI design
* 💰 Price display with clear visibility
* 📝 Meaningful product descriptions
* ➕ Add-to-cart button (UI level)
* 📱 Fully responsive (Mobile, Tablet, Desktop)
* ⚡ Smooth hover effects and transitions

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 📘 TypeScript
* 🎨 Tailwind CSS
* ⚡ Vite / Create React App

---

## 📂 Project Structure

```
src/
│
├── assets/        # Images and static files
├── components/    # Reusable components
├── pages/         # Page components (Home)
├── App.tsx        # Main app component
├── main.tsx       # Entry point
```

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/product-listing-app.git
```

### 2. Navigate to the project folder

```
cd product-listing-app
```

### 3. Install dependencies

```
npm install
```

### 4. Run the development server

```
npm run dev
```
